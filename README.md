# 🚀 InnovateX

InnovateX is a React-based web application developed to help students discover hackathons through a modern, responsive, and user-friendly interface. The platform provides organized information about hackathons while demonstrating modern frontend development using reusable React components.

## Project Overview

Many students struggle to find hackathons that match their interests because information is scattered across multiple platforms. InnovateX provides a centralized platform where users can browse hackathons through an intuitive interface. The application emphasizes responsive design, reusable components, and smooth navigation to improve the overall user experience.

## Features

- Browse hackathon opportunities
- Responsive user interface
- Component-based React architecture
- Easy navigation
- Modern frontend design
- Mobile-friendly layout

## Tech Stack

- React.js
- JavaScript
- HTML5
- CSS3

2. React components dynamically display different sections of the application.
